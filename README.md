# kermitco
kermitco is a project meant to allow special cases of "off-topic" learning. For anyone who is bored in school, this is place to go.

## Active instance
None yet!

## Installation guide
Clone the repo, then upload to a hosting site like GitHub pages or Cloudflare pages.
